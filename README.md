# 🚖 RideGo – Modern Ride Booking Website

RideGo is a modern and responsive ride-booking website inspired by popular cab service platforms like Uber.  
This project is built using **HTML**, **Tailwind CSS**, and **Vanilla JavaScript**.

It includes a clean UI, responsive layouts, animated sections, booking interactions, mobile menu, and modern design elements suitable for frontend portfolio projects.

---

## ✨ Features

- 🚗 Modern Uber-inspired UI
- 📱 Fully Responsive Design
- 🎨 Tailwind CSS Styling
- ⚡ Smooth Animations & Transitions
- 📍 Ride Booking Form
- 🚘 Ride Options Selection
- 📲 Mobile Navigation Menu
- 🌍 Explore Cities Section
- 🔒 Safety Features Section
- 👨‍✈️ Driver Signup Section
- 📥 App Download UI
- 🌙 Clean Dark/Light Styled Sections
- 💡 Interactive JavaScript Components
